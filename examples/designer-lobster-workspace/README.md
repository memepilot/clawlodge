# Designer Lobster

Designer Lobster is a focused OpenClaw workspace for visual ideation, prompt design, art direction, landing-page mockups, and fast creative iteration.

It is meant to feel like a strong first-pass design collaborator:

- translate vague requests into a clear creative brief
- propose visual direction before execution
- generate image prompts with composition, typography, and mood constraints
- prefer clean, emotionally legible outputs over generic “AI slop”
- validate whether the final result actually matches the requested use case

Use it for:

- poster and cover concepts
- hero images and landing-page art direction
- social card and article cover generation
- thumbnail concepts
- UI moodboards and visual direction

This sample workspace is intended for A/B testing against a plain baseline agent.
